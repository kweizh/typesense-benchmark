## `/home/user/import-pipeline/pipeline.py`

### Architecture

```
load_documents()
       │
       ▼
bulk_import(all docs, dirty_values=coerce_or_reject)   ← first pass
       │
       ├─ success → imported_first_pass
       │
       └─ failure → try_repair(doc)
                        │
                        ├─ repairable → bulk_import(repaired docs) → recovered / still-failed
                        │
                        └─ unrepairable → permanently failed
```

### Key design decisions

| Concern | Approach |
|---|---|
| **Idempotency** | Collection is dropped and re-created at the start of every run |
| **dirty_values** | `coerce_or_reject` — Typesense coerces compatible mismatches (e.g. `"45.50"` → `45.5`) but surfaces genuine errors as per-document failures |
| **Response parsing** | Every result line is mapped back to its source document by position; HTTP 200 is never trusted as "all succeeded" |
| **Repair R1** | `re.sub(r"[^\d.]", "", price_string)` strips currency symbols and commas; applied only when `price` is a non-numeric string |
| **Repair R2** | `category` set to `"uncategorized"` when the key is absent |
| **Server start** | Pipeline auto-starts `typesense-server` if not already reachable |
