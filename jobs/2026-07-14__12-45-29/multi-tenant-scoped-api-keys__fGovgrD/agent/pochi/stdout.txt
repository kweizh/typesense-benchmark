You must guarantee **hard multi-tenant isolation**: when a tenant searches, they may only ever see their own documents, and they must never be able to widen their access — even if they tamper with the search request client-side. Typesense solves this with **Scoped Search API Keys**: starting from a search-only parent key, you cryptographically embed a `filter_by` (and other search parameters) into a derived key. Typesense automatically enforces the embedded parameters on every search made with that key, and the caller cannot override them.
